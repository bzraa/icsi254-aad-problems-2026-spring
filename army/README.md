# Арми

Арми зүүн ба баруун гэсэн 2 фронтод хориглон байлдаж байгаа бөгөөд нийт N 
цэрэгтэй. Цэрэг бүр аль нэг фронтод байлдаж байгаа. Мөн тэд 1, 2, ..., 15
дугаартай 15 төрлийн чадвараас сонгон суралцсан байдаг. Цэргүүдийн аль фронт
дээр байгаа болон ямар чадварт суралцсан тухай мэдээлэл цэрэг бүрийн хувьд
16 тэмдэгтээр өгөгдсөн. Эхний тэмдэгт L (зүүн) эсвэл R (баруун) үсэг байх ба энэ
нь тухайн цэргийн байгаа фронтыг заана. Дараагийн тэмдэгтүүд 0 эсвэл 1 цифр байх
ба энэ нь дарааллаараа харгалзан 1, 2, ..., 15 чадваруудыг эзэмшсэн эсэхийг
илэрхийлнэ. Зүүн фронтын дайралт ширүүсэж байгаа тул армийн командлагч зүүн
фронтын хүчийг нэмхээр хүчийг нэмэхээр шийдвэрлэсэн. Харилцаа холбооны
хүндрэлээс үүдэн армийн командлагч зөвхөн нэг төрлийн команд дамжуулах боломжтой
байгаа. Уг команд нь ямар нэг чадварын дугаарыг фронтууд руу зэрэг илгээх бөгөөд
цэргүүд тухайн дугаартай чадварыг эзэмшсэн бол нөгөө фронт руу очих буюу фронтоо
солидог. Армийн командлагч энэ командыг хэдэн ч удаа ямар ч чадварын дугаар
сонгон илгээх боломжтой бол түүнд зүүн фронтдоо хамгийн олондоо хэдэн цэрэг
байрлуулж чадахыг тооцоолж өгнө үү.

## Оролт
- Эхний мөрөнд тестийн тоо T
- Дараагийн мөрөнд N тоо
- Дараагийн N мөр бүрт цэргүүдийн байрлал, чадварыг илэрхийлэх тэмдэгтүүд
## Гаралт
T мөр бүрт харгалзах хариу байна
## Хязгаарлалт
1 <= T <= 5, 1 <= N < 10^3
